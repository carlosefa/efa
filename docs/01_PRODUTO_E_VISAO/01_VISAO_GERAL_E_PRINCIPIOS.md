# Visão Geral e Princípios — EFA Esports

## O que é a EFA
Plataforma global para organizar campeonatos online (múltiplos jogos) com hierarquia de papéis, formatos flexíveis, estatísticas, rankings e auditoria.

## Princípios (guardrails)
1) Confiança acima de velocidade (log + motivo obrigatório para decisões).
2) Escala global (15 idiomas + timezones + políticas por país/org).
3) Economia opcional (Coins via feature flag; sistema funciona sem).
4) Sem pay-to-win (gamificação cosmética).
5) Marketplace com regras mínimas (organizador assume responsabilidade).
6) Dados imutáveis críticos (ledger/eventos append-only).
7) Configuração versionada (ruleset por torneio).

## MVP seguro
- Torneios (liga/mata-mata/grupos) com times pares/ímpares.
- Match room: reporte → confirmação → disputa → decisão admin com motivo.
- Rankings com mínimo de jogos + incerteza.
- Amistosos separados do competitivo.
- Páginas públicas fortes (torneio/time/jogador/org).
