# Calendário Global e Timezones

Regras:
- Persistir em UTC.
- País ativado já recebe timezone IANA default.
- Torneios internacionais escolhem “GMT oficial” antes de abrir.

Planner por janelas:
- allow_days (1..7)
- janela local (start/end)
- duração + buffer
- spacing por rodada
- blackout windows por time (opcional)
