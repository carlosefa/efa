# Critérios de Aceite e Casos de Uso

Critérios:
- motivo obrigatório em edições/decisões admin
- torneios suportam N ímpar (BYE)
- torneio internacional exige GMT oficial
- ranking: mínimo de jogos + incerteza
- coins e OCR só com feature flag

Casos:
- criar torneio
- inscrever
- reportar/confirmar
- abrir/decidir disputa
- amistoso internacional com GMT
- troféus e histórico
